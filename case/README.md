# Skeleton104 Case

## Top and Bottom Parts
- Top part (plate) mounts the switches.
- Bottom part (case) houses the PCB. 
- Both parts fit together to form the complete keyboard enclosure.

## Full and Split Parts
- `x_full` are single piece files intended for large format 3D printers or for users who prefer to split the 3D files themselves.
- `x_left` and `x_right` are split versions intended for standard size 3D printers.

## File Formats
Both STL and SLDPRT files are provided.
- STL files are provided for 3D printing.
- SLDPRT files are provided for users who want to modify the design in SolidWorks.
